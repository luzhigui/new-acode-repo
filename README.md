# 光明顶 5v5 对战

版本: V4.0.0 | 更新: 2026-06-29

类型: 5v5 回合制自走棋对战（明教 vs 六大派）| 九宫格站位、自动战斗、海克斯 Buff、精英技能

## 快速开始

直接用浏览器打开 `mode-5v5-test.html` 即可开始游戏。

## 目录结构

```
core/       - 战斗引擎核心（配置、单位、伤害计算、Buff、战马、引擎）
player/     - 战斗播放器（文本、Buff UI、核心、主播放器）
ui/         - UI 主控（工具函数、主控模块、渲染模块）
fx/         - 特效系统（伤害数字、箭头、飞撞、换位、击退、闪避、血斩、格挡）
modules/    - 功能模块（精英技能、错误捕获、音频管理）
tests/      - 测试与体检（单元测试、自动战斗、体检规则、体检核心、体检UI、答题、运行时采样）
tools/      - 开发工具箱（工具箱页面、工具箱JS、自动战斗工具、构建脚本）
```

## 测试与体检

打开 `tests/30test-runner.html` 进入测试与诊断中心，支持：
- 5个自动化测试
- 全面体检（约70条规则，12个分组）
- 25道答题模式

## 开发工具箱

打开 `tools/31-toolkit.html` 使用开发工具箱，包含：
- 文件复制器（大文件自动分片）
- 函数替换器
- 急救包（扫描经典Bug）
- 防战计算器
- 自动批量战斗
